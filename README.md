# Food-delivery 🍔🚀

A React-based **Food Delivery App** for a seamless and delightful food ordering experience.

## 📦 Tech Stack

- **React.js**
- **HTML5 & CSS3**
- **JavaScript (ES6+)**
- **Tailwind CSS**
- **Git & GitHub**

## ⚙️ Installation

To run this project on your local machine:

```bash
git clone https://github.com/apexx-0911/Food-delivery.git
cd Food-delivery
npm install
npm start
